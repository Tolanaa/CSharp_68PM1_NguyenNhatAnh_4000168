## Mô tả dự án
Ứng dụng Windows Forms C# thực hiện chức năng **Đăng Nhập**.

## Chức năng
- Màn hình đăng nhập với giao diện đẹp
- Kiểm tra tên đăng nhập = Email sinh viên
- Kiểm tra mật khẩu = MSSV
- Hiển thị MessageBox thành công / thất bại

## Công nghệ
- **Ngôn ngữ:** C#
- **Framework:** .NET 6 / Windows Forms
- **IDE:** Visual Studio 2022

## Cách chạy
1. Mở file `DangNhap.csproj` bằng Visual Studio 2022
2. Nhấn `F5` hoặc `Ctrl+F5` để chạy
3. Nhập email sinh viên và MSSV để đăng nhập
